# EV 에어로다이내믹 시뮬레이터
 
전기차의 주행 속도에 따른 공기저항과 예상 전비를 실시간으로 계산해주는 인터랙티브 시뮬레이터입니다. 결측치, 이상치, 피처 선택을 물리 공식과 함께 시각적으로 체험할 수 있도록 만들었습니다.
 
## 🔍 주요 기능
 
- **실시간 물리 연산**: 전고·전폭·속도 값을 바꾸면 전면 투영 면적(A), 공기 저항력(Fd), 구름 저항(Fr)이 즉시 재계산되어 화면에 반영됩니다.
- **AI 제어 모드 시뮬레이션**: 속도 구간(60km/h, 100km/h)에 따라 냉각 플랩·스포일러 등 제어 로직이 COOLING / BALANCED / ECO-AERO 모드로 전환됩니다.
- **차량 프리셋**: 아이오닉 6, 테슬라 모델 3의 실제 공력계수(Cd)를 적용해볼 수 있습니다.
- **데이터 전처리 시나리오 체험**
  - 이상치(Outlier) 데이터 로드 → IQR 기준 초과 시 경고 및 슬라이더 잠금
  - 결측치(Missing Value) 시뮬레이션 → 중앙값 대체 로직 확인
  - 상관관계가 낮은 피처(전장)를 실제로 제외하고, 값을 바꿔도 결과가 변하지 않는 것을 직접 확인
- **모델 성능 리포트**: 회귀/분류 모델의 R², MSE, 정확도 등 학습 결과를 표로 함께 제공
- **수식 실시간 대입**: MathJax로 렌더링된 공식에 실제 대입값이 실시간으로 표시됩니다.
## 🛠 기술 스택
 
- Vanilla HTML / CSS / JavaScript (프레임워크 없음)
- MathJax 3 (수식 렌더링)
