# ⚡ EV 에어로다이내믹 시뮬레이터
 
> 전기차의 **주행 속도 · 차체 크기**를 조절하면, 공기저항과 예상 전비가
> 실시간으로 계산되어 화면에 반영되는 인터랙티브 물리 시뮬레이터입니다.
 
전기차의 전비는 공기저항의 영향을 크게 받지만, 그 원리(공식·변수)를 직접
눈으로 확인하기는 어렵습니다. 이 서비스는 전고·전폭·속도 값을 슬라이더로
조절하면 공기 저항력(Fd)과 예상 전비가 어떻게 바뀌는지 물리 공식과 함께
실시간으로 보여주며, 동시에 **이상치·결측치·피처 선택** 같은 데이터
전처리 개념을 함께 체험할 수 있도록 만들었습니다.
 
---
 
## ✨ 주요 기능
 
- **실시간 물리 연산** — 전고·전폭·속도 값을 바꾸면 전면 투영 면적(A),
  공기 저항력(Fd), 구름 저항(Fr)이 즉시 재계산되어 수식과 함께 화면에
  반영됩니다.
- **AI 제어 모드 시뮬레이션** — 속도 구간(60km/h, 100km/h)에 따라 냉각
  플랩·스포일러 등 제어 로직이 COOLING / BALANCED / ECO-AERO 모드로
  자동 전환됩니다.
- **차량 프리셋** — 아이오닉 6, 테슬라 모델 3의 실제 공력계수(Cd)를 적용해
  비교해볼 수 있습니다.
- **데이터 전처리 시나리오 체험**
  - 이상치(Outlier) 데이터 로드 → IQR 기준 초과 시 경고 및 슬라이더 잠금
  - 결측치(Missing Value) 시뮬레이션 → 중앙값 대체 로직 확인
  - 상관관계가 낮은 피처(전장)를 실제로 제외하고, 값을 바꿔도 결과가
    변하지 않는 것을 직접 확인
- **모델 성능 리포트** — 회귀/분류 모델의 R², MSE, 정확도 등 학습 결과를
  표로 함께 제공합니다.
- **수식 실시간 대입** — MathJax로 렌더링된 공식에 실제 대입값이 실시간
  으로 표시됩니다.
### 시뮬레이터에서 확인할 수 있는 물리 공식
 
| 항목 | 공식 |
|------|------|
| 전면 투영 면적 | $A = 전고 \times 전폭$ |
| 공기 저항력 | $F_d = \frac{1}{2} \rho v^2 C_d A$ |
| 주행 저항 성분 | $F_{total} = F_d(공기저항) + F_r(구름저항)$ |

## 🗂 프로젝트 구조
 
```
.
├── index.html      # 메인 페이지 (UI 및 수식 표시 영역)
├── css/
│   └── style.css   # 디자인·스타일
└── js/
    └── script.js   # 물리 연산 및 인터랙션 로직
```
 
이 서비스는 **별도의 서버나 빌드 과정이 필요 없는** 순수 정적 웹사이트입니다.
HTML/CSS/JavaScript만으로 동작하므로 누구나 쉽게 열어보고 수정할 수 있습니다.

## 🚀 실행 방법
`index.html` 파일을 웹 브라우저로 열기만 하면 됩니다.
 
---
 
## 🛠 수치·시나리오 수정하기
 
기본값이나 시나리오(이상치 기준, 중앙값 대체값 등)를 바꾸고 싶다면
**`js/script.js` 상단의 상수 값**만 수정하면 됩니다.
 
- `DEFAULT_WIDTH_MM`, `DEFAULT_LENGTH_MM`, `DEFAULT_MASS` → 기본 차량 제원
- `MEDIAN_HEIGHT_MM` → 결측치 대체에 쓰이는 중앙값
- `OUTLIER_WIDTH_M` → 이상치 데이터 로드 시 적용되는 전폭 값
---
 
## ⚠️ 안내
 
- 시뮬레이터에 사용된 공기 밀도(ρ), 구름저항계수(Cr) 등은 **고정값을 사용한
  단순화된 물리 모델**이며, 실제 차량의 전비와는 차이가 있을 수 있습니다.
- 모델 성능 표(R², 정확도 등)는 예시 데이터를 학습한 결과이며, 실제 서비스에
  적용하기 위한 지표가 아닙니다.
